==> Modo Iterativo Python
Serve para que possamos efetuar cálculos via python de forma
iterativa sem necessidade de iniciar a ide.
Para iniciar é bem simples, basta digitar o comando abaixo no terminal:
python3 ou python
A depender da versão do python instalado no computador;
Alguns comandos podem ser utilizado no modo iterativo,
iremos conhecer o dir e help.

-> dir()
Sem argumentos, retorna a lista de nomes no escopo local atual.
Com um argumento, retorna uma lista de atributos válidos para o objeto.
exemplo
dir()
dir(100)

-> help()
Invoca o sistema de ajuda integrado. É possível fazer buscas em modo interativo
ou informar por parâmetro qual o nome do módulo, função, classe, método
ou variável. Exemplo:
help();
help(100);
